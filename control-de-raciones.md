# Política de Privacidad - Control de Raciones

**Última actualización:** Mayo 2026

**Desarrollador:** ZenisApps

## 1. Información que recopilamos

Control de Raciones recopila la siguiente información para funcionar correctamente:

- Número de serie del usuario (identificador interno)
- Selecciones de comidas diarias
- Departamento y edificio asignado
- Token de notificaciones push para envío de alertas
- Usamos Google AdMob para mostrar anuncios, que puede recopilar un identificador publicitario del dispositivo

## 2. Uso de la información

La información recopilada se usa exclusivamente para:

- Gestionar el sistema de control de raciones de la empresa
- Enviar notificaciones sobre comidas disponibles o cambios en el menú
- Mostrar anuncios relevantes a través de Google AdMob

## 3. Almacenamiento de datos

Todos los datos se almacenan de forma segura en Firebase (Google Cloud). No vendemos ni compartimos tu información con terceros.

## 4. Servicios de terceros

- **Firebase (Google):** Autenticación y base de datos. Política: https://firebase.google.com/support/privacy
- **Google AdMob:** Monetización. Política: https://policies.google.com/privacy
- **Expo:** Notificaciones push. Política: https://expo.dev/privacy

## 5. Seguridad

Implementamos medidas de seguridad para proteger tu información. El acceso está restringido por número de serie y contraseña.
