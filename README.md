# Política de Privacidad - Calculadora de Subredes

**Última actualización:** Mayo 2026

## 1. Información que recopilamos
Esta aplicación no recopila información personal identificable. Utilizamos Google AdMob para mostrar anuncios, que puede recopilar un identificador publicitario del dispositivo.

## 2. Uso de la información
La información recopilada por AdMob se usa exclusivamente para mostrar anuncios relevantes. No vendemos ni compartimos tu información con terceros.

## 3. Servicios de terceros
Usamos Google AdMob para monetización. Su política de privacidad está disponible en: https://policies.google.com/privacy

## 4. Seguridad
No almacenamos datos personales en nuestros servidores. Todos los cálculos se realizan localmente en tu dispositivo.
